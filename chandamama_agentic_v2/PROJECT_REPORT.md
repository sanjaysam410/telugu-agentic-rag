# Chandamama Agentic Engine v2: Project Report

## 1. Executive Summary
This project ("Chandamama Agentic Engine") is an advanced data ingestion and validation pipeline for a Telugu children's content archive. The system uses LLMs (specifically `openai/gpt-oss-120b` via Groq) to validate contributions and automatically generate rich metadata.

**Core Achievements:**
-   **Model Upgrade:** Transitioned to **Groq's 120b model**, enforcing a single highly capable open-source model across the stack.
-   **Content Inclusivity:** Relaxed validation logic to accept Poems, Songs, and Articles, expanding beyond just narrative stories.
-   **Rich Metadata:** Now supports native language Genre extraction (e.g., "జానపదం") and stores full textual context.
-   **UI Overhaul:** Replaced basic dropdowns with a modern **Tabbed Interface** and added comprehensive fields for archival quality (Source Year, Illustrator, etc.).

## 2. Key Features Implemented

### A. Single Model Enforcement (Groq)
-   **Engine**: `agentic_engine/llm_client.py` and `config.py` were updated.
-   **Goal**: Prevent accidental usage of expensive or fallback models.
-   **Implementation**: Removed `gpt-4o-mini` defaults. All calls now route through `DEFAULT_AGENT_MODEL` mapped to `GROQ_API_KEY`.
-   **Validation**: Verified via `tests/verify_groq.py` which confirmed successful API connection and token limits (raised to 2000).

### B. Story Validation Agent (Agent 1)
-   **Old Logic**: Strict narrative check; rejected anything without a clear "plot".
-   **New Logic**:
    -   Explicitly accepts **Poems, Songs, Cultural Articles**.
    -   Prompt revised to be "Permissive by Default" (only rejecting gibberish/hate speech).
    -   Token limit increased to 2000 to handle complex reasoning.

### C. Metadata Generation Agent (Agent 2)
-   **Native Language Support**:
    -   **Genre Extraction**: Instructed LLM to extract genre in the target language (e.g., Telugu).
    -   **Normalization**: Added a mapping layer in `_normalize_genre` to convert Telugu terms (e.g., "నీతి") to standard English codes (`MORAL_STORY`).
-   **Full Context Storage**: Added `full_text` field to the output JSON to ensure the metadata file is self-contained.
-   **Language Propagation**: Selected language in UI (e.g., "Hindi") is now passed to the agent, ensuring the summary and themes are generated in that language.

### D. UI & Experience (agentic_app.py)
-   **Navigation**: Switched from Sidebar Selectbox to **Streamlit Tabs** (`st.tabs`) for faster switching between Contribution, Metadata, and Workflow modes.
-   **New Fields**: Added `Author`, `Genre`, `Target Age`, `Source Year`, `Keywords`, `Illustrator`.
-   **Auto-Translation**: When a user selects a Genre in English (e.g., "Folklore"), the UI automatically translates it to the target language (e.g., "జానపదం") before submission.

### E. Sandbox Environment
-   **New Directory**: `chandamama_agentic_v2/`
-   **Purpose**: A completely isolated folder containing the modified App, Engine, Config, and Requirements. This ensures the original codebase remains untouched while allowing full functionality for the new version.
